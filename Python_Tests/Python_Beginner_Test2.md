#### Question 2

```python

```
a) `[1, 4, 3]`

c) `[4, 2, 3]`  
d) `TypeError`  
**Answer:** a) `[1, 4, 3]`  
**Explanation:** Lists in Python are mutable, so you can change their elements. The code changes the second element (index 1) to 4.
